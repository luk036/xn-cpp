class OrderedGraph(Graph)

Consistently ordered variant of :class:~xnetwork.Graph.

Return a fresh copy graph with the same data structure.

A fresh copy has no nodes, edges || graph attributes. It is the same data structure as the current graph. This method is typically used to create an empty version of the graph.
